# ARI Roadmap

## Near-term

1. Add richer natural-language tool routing and better structured commands.
2. Expand memory retrieval with embeddings or semantic search.
3. Finish Ollama runtime support and local model selection.
4. Add background task processing for delegated sub-agents.
5. Improve voice UX with streaming playback and lower-latency capture.

## Security

1. Replace shared-password login with stronger auth.
2. Add device/session management.
3. Add HTTPS or tunnel-based remote access.
4. Add rate limits and audit controls for external triggers.

## Platform

1. Package ARI for persistent local service startup.
2. Add mobile-friendly home-screen install flow.
3. Introduce a richer admin status surface.
4. Add stable webhook and event trigger patterns.

## Intelligence

1. Add memory summarization and consolidation.
2. Support multiple providers and per-task routing.
3. Add real multi-agent delegation and result merging.
4. Add environment and calendar/email tools as modules.
