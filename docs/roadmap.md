# Roadmap

## Current Focus

Stabilize the local-first execution spine.

This includes:

- canonical state
- persistence
- explainable orchestration
- hub visibility
- approval boundaries
- mobile surfaces

## Near-Term Goals

### 1. Stabilize Career Command

- improve scouting workflows
- improve ranking and evaluation
- improve pending approval handling
- improve execution tracking
- improve reporting and summaries

### 2. Expand ACE Surfaces

- improve hub visibility
- improve Telegram integration
- improve mobile awareness flows
- expose operational state more clearly

### 3. Strengthen Explainability

- improve orchestration visibility
- improve signal evidence paths
- improve alert inspection
- expose execution history cleanly

### 4. Build Inspection Cabinet

- durable introspection views
- orchestration inspection
- approval history
- memory inspection
- state inspection
- execution replay direction

### 5. Improve Operational Tooling

- safer orchestration paths
- stronger validation
- cleaner repository structure
- stronger testing
- reproducible local startup

## Longer-Term Exploration

Potential future directions include:

- multi-agent coordination
- advanced voice interaction
- broader automation tooling
- AI-assisted operational consulting
- construction and finance workflows
- content and research pipelines
- persistent ambient computing surfaces

## Non-Goals Right Now

The current focus is not:

- maximum autonomy
- uncontrolled agent behavior
- aggressive cloud scaling
- consumer-polished UI
- broad enterprise deployment

The focus is building a durable and explainable operational core first.
